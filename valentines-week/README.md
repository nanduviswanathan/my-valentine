A magical, interactive React application designed to celebrate Valentine's Week (Feb 7 - Feb 14). Each day unlocks a unique, animated experience dedicated to that specific day of love.

## Features ‚ú®

- **Rose Day (Feb 7)**: Interactive 3D Rose Gift.
- **Propose Day (Feb 8)**: Romantic proposal scene.
- **Chocolate Day (Feb 9)**: Unwrapping Ferrero Rocher chocolates.
- **Teddy Day (Feb 10)**: Cute animated teddy bear.
- **Promise Day (Feb 11)**: Interactive promise cards.
- **Hug Day (Feb 12)**: Warm hug simulation.
- **Kiss Day (Feb 13)**: Flying kisses animation.
- **Valentine's Day (Feb 14)**: The grand finale!
- **Memory Lane**: A post-Valentine's gallery to look back on memories.

## Getting Started üöÄ

Follow these steps to run the project locally on your machine.

### Prerequisites

- [Node.js](https://nodejs.org/) (Version 16 or higher recommended)
- npm (comes with Node.js) or yarn

### Installation

1.  **Clone the repository** (if you haven't already):
    ```bash
    git clone <repository-url>
    cd valentines-week
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```

### Running Locally

Start the development server:

```bash
npm run dev
```

Open your browser and navigate to the URL shown in the terminal (usually `http://localhost:5173`).

### Building for Production

To create a production-ready build:

```bash
npm run build
```

The output will be in the `dist` folder, ready to be deployed to Vercel, Netlify, or GitHub Pages.

## Navigation Tips üß≠

- The app automatically shows the content for the current date.
- Use the **Menu Button** (bottom right) to navigate to previous days you've unlocked.
- To **test specific days**, you can add a `?date=YYYY-MM-DD` query parameter to the URL:
    - Example: `http://localhost:5173/?date=2026-02-14`
